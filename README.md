# Yayasan Surabaya Mengaji

Yayasan Surabaya Mengaji adalah lembaga dakwah dan pendidikan Islam yang berfokus pada pengembangan program keislaman, pendidikan, sosial, dan pelayanan umat.

Melalui pemanfaatan teknologi, kami mengembangkan berbagai platform digital untuk mendukung operasional yayasan, pelayanan donatur, pengelolaan program, serta penyebaran manfaat kepada masyarakat.

## Our Mission

- Menyebarkan nilai-nilai Islam yang rahmatan lil 'alamin.
- Mendukung kegiatan dakwah dan pendidikan Islam.
- Mengelola amanah donasi secara profesional dan transparan.
- Mengoptimalkan teknologi untuk meningkatkan pelayanan umat.

## Technology

Kami menggunakan berbagai teknologi modern untuk membangun dan mengelola layanan digital yayasan, termasuk:

- TypeScript
- Node.js
- NestJS
- React
- Next.js
- PostgreSQL
- Docker
- Cloudflare
- GitHub Actions

## Development

Repository dalam organisasi ini digunakan untuk:

- Website dan landing page
- Portal administrasi
- Sistem manajemen donatur
- Integrasi pembayaran dan laporan
- Layanan digital pendukung program yayasan
